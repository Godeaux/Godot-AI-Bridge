# Godot AI Bridge

A Godot 4.6 addon + MCP server that gives AI agents full control over both the Godot editor and running games.

See [`godot_ai_bridge/README.md`](godot_ai_bridge/README.md) for full documentation and setup instructions.

## Quick Start

1. Copy `godot_ai_bridge/addons/godot_ai_bridge/` into your Godot project's `addons/` folder
2. Enable the plugin in **Project > Project Settings > Plugins**
3. `pip install fastmcp httpx`
4. Configure your AI client's MCP settings (use the **Copy MCP Config** button in Godot's AI Bridge panel)
5. Ask your AI agent to interact with your Godot project
